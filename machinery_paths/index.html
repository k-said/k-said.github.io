📖 Machinery Path Planner — Help

🚀 Getting Started

1. Draw or load a field boundary
Click ⚙ Parameters to open the sidebar, then either:
• Click Draw polygon on map and draw your field boundary directly, or
• Scroll down and use Load boundary.geojson to upload an existing file.

2. Place the gate (entry point)
Click Click map to place gate, then click on the map where the machine enters/exits the field.

3. Set parameters
Adjust Lane width, Headway width, etc. in the sidebar. For contour‑following mode, enable Use elevation contours under Advanced parameters.

4. Generate paths
Click the green ▶ Generate paths button. The server computes lanes, headways, route, and statistics. Results appear on the map and in the 📊 Path Statistics panel (bottom‑left).

5. Animate & inspect
Use ▶ Play to animate the tractor along the smoothed route. Toggle layers with the checkboxes in the top bar. Switch the map background between Imagery, Map, or None.

🌾 Main Parameters

Lane width (m) — distance between adjacent working lanes (centre to centre). Should match your implement working width so coverage is complete without excessive overlap.
Default: 10 m
Headway width (m) — distance between successive headway rings (the concentric paths around the inside of the field boundary). Matches implement width for the headway pass.
Default: 5 m
Headway-border gap (m) — offset from the field boundary to the first (outermost) headway ring. Controls how far inward the tractor drives from the field edge.
Default: 5 m
Num headways — how many concentric headway rings to create inside the boundary before generating interior lanes. More headways = more room for turning at the field edges.
Default: 2
Lane orientation (°) — angle (degrees from east, counter-clockwise) of the interior lanes. Leave blank for auto (picks the orientation that best fits the field shape, minimising the number of short lanes).
Headways first — when checked, the tractor drives the headway rings before entering the interior lanes. When unchecked, lanes are driven first and headways last.
Gate (entry point) — click on the map to place the field entry/exit point. The route starts and ends at this point. If omitted, the first vertex of the boundary is used.

🔧 Advanced Parameters

Lane-headway gap (m) — extra clearance between the innermost headway ring and the start/end of the interior lanes. Use this to avoid collisions with headway traffic.
Default: 0 m
Lane boundary offset (m) — additional inward offset applied to the boundary before clipping lanes. Prevents the implement from touching the boundary edge.
Default: 0 m
Smooth angle tol. — angle threshold (radians) below which corners in the route are smoothed with circular arcs.
Default: 0.5

Smooth segments — number of arc segments used to approximate each smoothed corner.
Default: 8

Smooth tol. dist. (m) — max distance the smoothed arc may deviate from the original corner point.
Default: 5 m

Smooth simplify (m) — Douglas-Peucker simplification tolerance applied to each route segment before smoothing.
Default: 2 m

🗻 Elevation Contour Parameters

Use elevation contours — when enabled, lanes follow terrain contours instead of straight parallel lines. This reduces erosion risk on sloped fields and can lower fuel consumption.

Contours gap (m) — vertical elevation difference between adjacent contour lines extracted from the DEM.
Default: 1 m

Contour levels — number of contour levels. Leave blank for auto (derived from elevation range ÷ contours gap).

🌱 Sustainability Inputs

Fuel rate (L/km) — fuel consumed per km of travel.
Default: 0.35

Fuel price (€/L) — cost per litre of fuel.
Default: 1.50

CO₂ per L fuel (kg) — kg of CO₂ emitted per litre of fuel burned.
Default: 2.65

Machine weight (t) — total weight of tractor + implement (tonnes). Used for ground-pressure / compaction estimation.
Default: 8 t

Tyre width (m) — width of one tyre. Used to compute tyre-track coverage area and soil compaction.
Default: 0.6 m

Speed (km/h) — average field speed. Used to estimate total field time.
Default: 8 km/h

🎮 Viewer Controls

Background — switch between satellite Imagery, Map (OpenStreetMap), or None (dark background).

Checkboxes — toggle layer visibility on the map: Lanes+Headways, Smoothed route, Boundary, Contours, Selected contour.

Play / Pause — animate the tractor along the smoothed route.
Speed slider — animation playback speed.
Progress bar — scrub through the animation.

Reset View — clears generated paths but keeps the boundary.
Reset All — clears everything including the boundary and gate.
